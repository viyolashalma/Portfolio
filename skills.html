HTML

HTML (HyperText Markup Language) is the standard markup language used to create and design the structure of web pages. It defines the basic building blocks of web content and allows browsers to display text, images, forms, and other elements on the web.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a web page. It allows you to control the layout, colors, fonts, spacing, and overall visual appearance of HTML elements.

JavaScript

JavaScript is a versatile, high-level programming language that is primarily used to create interactive and dynamic content on web pages. It enables websites to go beyond static pages and become fully interactive.

Node.js

Node.js is an open-source, cross-platform JavaScript runtime environment that allows developers to execute JavaScript code outside of a web browser. It is commonly used to create scalable, high-performance web applications and APIs.

React.js

React.js is a JavaScript library for building user interfaces. It allows developers to create reusable UI components, manage state efficiently, and render components dynamically based on user interaction.

Python

Python is an interpreted, object-oriented, high-level programming language with dynamic semantics developed by Guido van Rossum. It is designed to be easy to understand and fun to use.

C++

C++ is an object-oriented programming (OOP) language that is widely used for creating large-scale applications. It is a superset of the C language, providing better tools for system-level programming.

MySQL

MySQL is the world’s most popular open-source database. It powers many of the most accessed applications, including Facebook, Twitter, Netflix, Uber, Airbnb, and more.

MongoDB

MongoDB is a NoSQL database that utilizes JSON-like documents with optional schemas. It is highly scalable and is used in modern applications where flexibility in database structure is required.

C

The C programming language is a general-purpose language that supports structured programming and provides low-level access to memory, making it ideal for system-level programming.

MUI

MUI (Material-UI) is a popular React UI framework that offers a wide range of pre-built components based on Google’s Material Design, helping developers build consistent, modern, and responsive web applications.

Canva

Canva is an online graphic design platform that allows users to create social media graphics, presentations, posters, and other visual content using an easy-to-use drag-and-drop interface and templates.

Photoshop

Adobe Photoshop is a powerful image editing software used by professionals to create, enhance, and manipulate images and graphics for both digital and print media.

InShot

InShot is a user-friendly mobile video and photo editing app that enables users to trim, merge, add effects, and apply filters, ideal for creating content for social media platforms.

DBMS

A Database Management System (DBMS) is software that enables the creation, storage, modification, and retrieval of data in databases, ensuring data integrity, security, and efficient access for users and applications.
